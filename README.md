รวบรวมฟังก์ชั่นเล็กๆน้อยๆใน vfp ที่ผมสร้างเพื่อใช้กับงานต่างของตัวเองและลูกค้า เพื่อทำให้จบงานง่ายขึ้น มีทั้ง
1. แบบช่วยเล็กๆน้อยๆเพื่อให้คำสั่งมันเรียบง่ายขึ้นเวลาอ่านโค้ด 
หรือ 2. บางครั้งไม่อยากต้องมาจดจำและใส่ parameter ของฟังก์ชั่นมาตรฐาน vfp ซ้ำๆเนื่องจากบาง parameter ก็ไม่เคยได้ใช้งานเลย หลายๆครั้งจำไม่ได้เพราะเขียนหลายภาษาแล้วสับสน 
และ 3.แบบที่ไม่มีฟังก์ชั่นที่ทำงานแบบนั้นเลยใน vfp เลยสร้างขึ้นมาใช้เพื่อเสริมให้ vfp สามารถทำงานบางอย่างได้ บางครั้งก็เป็นแบบกำปั้นทุบดิน เอาสีข้างเข้าถู
